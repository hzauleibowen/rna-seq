## 1.数据下载
在linux里使用后台命令进行下载，虽然上传的wget文件里都有，但是下载的少了两个，利用R语言的setdiff函数比对原始数据和下载的数据就可以很快找到缺失的两个数据。

```
nohup wget -O "NAME" -c site>NAME.out &  #download 
```
